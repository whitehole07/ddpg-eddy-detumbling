## **1. Introduction**
This repository contains the code, models, and analysis developed for my master's thesis on **space debris detumbling using eddy currents and reinforcement learning (RL)**. The research investigates the viability of **robotic manipulators with electromagnetic end-effectors** to stabilize debris in orbit, leveraging the **Deep Deterministic Policy Gradient (DDPG) algorithm** for optimal control.
